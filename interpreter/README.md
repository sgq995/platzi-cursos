# Interpreter
A interpreter for basic dynamic programming language.

## Syntax
The following is an example of the program:
```
var a = 1;
var b = 2;

var list = [];
var dict = { "key": "value" };

var sum = func(a, b) {
  return a + b;
}

var test = func(num, limit) {
  if (num >= limit) {
    return true;
  } else {
    return false;
  }
}
```
